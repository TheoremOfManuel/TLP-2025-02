Es importante mencionar que el proyecto para esta altura esta realizado en Python 3, por tanto, para ejectarlo basta con en la temrinal llamar a "python" y ejecutar el runtime. Es decir, > Python runtime.py < de esta manera se ejecuta. Pero el para el proyecto que tiene el profe que se realizo con Python 27 lo que hay que irse al lugar donde esta y luego <.\ python.exe> para decirle a Windows que necesito el ejecutador de Windows y leugo poner la ruta hasta el runtime seguido de la ruta para alguno de los archivos .json.
